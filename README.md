# Hytale UI IDEA Plugin
A Jetbrains plugin to provide better support for Hytale UI files.

## Features
- Parsing and validation of Hytale UI files, any errors are shown in the editor
- Syntax highlighting
- Formatting (indentation + spacing)
- Code completion
  - Element types
  - Element properties
  - Object properties
  - Variable names
- Code folding
- Quick documentation
  - Variable value
  - Element properties
  - Object properties


## Future Improvements
- Partial parsing of UI files with parser errors to avoid losing editor features
